## Featured Projects

### 🔷 PortSense — AI-Powered Portfolio Analytics
An investment analytics platform for Indian retail investors featuring live P&L tracking, FinBERT-based sentiment analysis on portfolio holdings, Nifty 50 benchmark comparison, diversification scoring, and Gemini-powered rebalancing recommendations.

**Stack:** React · FastAPI · MongoDB Atlas · FinBERT · Gemini API · yfinance  
[🔗 Live Demo](https://port-sense-tau.vercel.app/) · [GitHub](https://github.com/Dev-06-06/PortSense)

---

### 💰 SmartTax — ML-Powered Micro-Savings & Bank Analytics Platform
A fintech platform that converts everyday UPI transactions into goal-based savings using ML-driven recommendations, while providing banks with actionable customer insights through Power BI dashboards. Features a full ETL pipeline, Prophet-based spending forecasts, KMeans customer segmentation, and a dual-sided architecture serving both retail users and bank analysts.

**Stack:** React · Flask · Python · MongoDB Atlas · Prophet · Scikit-learn · Power BI  
[GitHub](https://github.com/Dev-06-06/SmartTax)

---

### 🏏 CricTrack — Real-Time Cricket Scoring App
A full-stack cricket match management app with live ball-by-ball scoring, group-based player management, and real-time score sync across clients using WebSockets.

**Stack:** React · Node.js · Express · Socket.IO · MongoDB · TailwindCSS  
[🔗 Live Demo](https://cricket-tracker-rosy.vercel.app/) · [GitHub](https://github.com/Dev-06-06/cricket-tracker)
